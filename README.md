# Veille Presse Procivis

App Streamlit de consultation de la veille presse hebdomadaire Procivis.
Alimentée automatiquement par un workflow N8N qui analyse les PDF via Claude.

## Déploiement sur Streamlit Cloud

1. Connecter ce repo dans [share.streamlit.io](https://share.streamlit.io)
2. Fichier principal : `app.py`
3. Dans **Settings > Secrets**, coller le contenu de `secrets_template.toml` avec vos vraies valeurs
4. Déployer
